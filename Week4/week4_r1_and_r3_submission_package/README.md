# Week 4 R1 + R3 Submission Package

This package contains the files for the user's Week 4 responsibilities:

- `product_owner.md` — R1 Product Owner output
- `almanac.md` — R3 Almanac Agent output
- `DECISION.md` — Architecture decision paragraph required for Sprint 4
- `prediction_2026-W24_R1_R3_draft.md` — Draft prediction using R1/R3 evidence
- `assets/` — charts used by the R3 file

## Upload suggestion

Place these files in the GitHub repo under a Week 4 folder, for example:

```text
evidence/2026-W24/
├── almanac.md
├── product_owner.md
├── prediction_2026-W24_R1_R3_draft.md
├── DECISION.md
└── assets/
```

If the team already has a root-level `DECISION.md`, merge the content instead of overwriting other members' work.
